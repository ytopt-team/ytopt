This directory includes all files for autotuning ECP app [HeFFTe](https://github.com/icl-utk-edu/heffte) using ytopt-libe on a laptop, ALCF Theta and OLCF Crusher.

# Directory

```
heffte/
    Source code for HeFFTe (download from https://github.com/icl-utk-edu/heffte)
laptop/	
    Autotuning HeFFTe on a laptop for demonstration
ytopt/
    Autotuning HeFFTe using ytopt only on a laptop
theta/
    Autotuning HeFFTe on ALCF Theta
crusher/	
    Autotuning HeFFTe on OLCF Crusher 

```
